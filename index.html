Research problem background

How can artificial intelligence and big data, be used to predict poverty indicators ?

Over the last few decades, poverty has been measured using conventional surveys which are relatively expensive. Various studies continue to be developed and focus on the finding that big data, such as satellite imagery can be used to explain variations in poverty more comprehensively and can reach subpopulations that conventional surveys may not be able to reach.

{% comment %}

In this study, technology adoption is carried out by utilizing satellite imagery data and Convolutional Neural Network (CNN) with transfer learning to overcome the limitations of large amounts of training data. The CNN model is then used to extract meaningful features in each sub-district which are expected to be strongly correlated with local expenditures. This feature is then used as additional information in estimating the average per capita expenditure at the sub-district administrative level using Small Area Estimation (SAE).

{% endcomment %}

What is different from previous research ?

The following are some improvements from previous research, especially for the research of Jean et al. (2016)

  • Re-evaluate Jean et al (2016) framework at a smaller administrative level (sub-district).
  • The main estimation is done using a SAE not a machine learning regression algorithm.
  • Using the high-resolution NTL of the VIIRS sensor as the basis for labeling daylight images (not DMSP-OLS).
  • Build a CNN model with two different training data, namely pixel and area data.
  • Have used more than one pre-trained model architecture, such as MNetV2, MNetV3, ResNet50, InceptionV3, and VGG19.
